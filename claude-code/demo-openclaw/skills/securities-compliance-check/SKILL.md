# 证券合规预检 Skill

## 核心职责

检查营销话术、合同条款、客户沟通内容是否存在违规表述，输出结构化风险预检清单。

## 三条绝对红线

以下内容**一律拦截，不得输出**：

1. **不承诺收益**
   - 禁止词：保本、无风险、稳赚、保证收益、一定涨、稳定回报、100% 安全
   - 禁止句式："这个产品不会亏的""买了肯定赚"

2. **不代客操作**
   - 禁止行为：接受客户委托下单、代持账户、替客户做投资决策
   - 禁止句式："我帮你买""你把账号给我"

3. **不构成投资建议**
   - 超出知识库范围的，引导联系持牌投顾
   - 禁止句式："你应该买 XX 股票""我建议你全仓"

## 检查流程

1. **识别内容类型**
   - 营销话术 / 客户微信消息 / 合同条款 / 产品介绍材料

2. **逐条扫描**
   按以下维度检查：
   - 收益承诺类（高风险）
   - 风险误导类（中风险）
   - 适当性缺失类（中风险）
   - 表述不规范类（低风险）

3. **输出风险预检清单**

## 输出格式

```
## 合规预检报告

**检查对象**：[内容类型]
**检查时间**：[当前时间]
**整体风险等级**：🔴 高 / 🟡 中 / 🟢 低

### 风险条目

| 序号 | 原文摘录 | 风险类型 | 风险等级 | 整改建议 |
|------|---------|---------|---------|---------|
| 1 | "..." | 收益承诺 | 🔴 高 | 删除/替换为"... " |

### 合规结论
[通过 / 需整改后再用 / 禁止使用]

### 整改版本（如需要）
[给出合规的替换表述]

---
⚠️ 本报告由 AI 辅助生成，仅供内部参考，最终合规判断以合规部意见为准。
```

## 常见违规话术对照表

| 违规表述 | 风险等级 | 合规替换 |
|---------|---------|---------|
| 这个产品保本的 | 🔴 高 | 该产品为[风险等级]产品，投资有风险 |
| 年化收益稳定在 8% | 🔴 高 | 历史业绩不代表未来收益 |
| 买这个肯定不亏 | 🔴 高 | 删除，不得做任何收益承诺 |
| 我帮你操作一下 | 🔴 高 | 删除，不得代客操作 |
| 这只股票会涨 | 🔴 高 | 以下内容仅供参考，不构成投资建议 |
| 风险很小 | 🟡 中 | 该产品风险等级为 XX，请确认与您风险承受能力匹配 |
| 收益还不错 | 🟡 中 | 过往收益为 XX%，不代表未来表现 |

## 使用限制

- 仅检查文字内容，不核实数据真实性
- 不替代合规部门的正式审核
- 每次检查结果须人工复核后使用
