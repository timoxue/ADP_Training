# 证券行业金融营销禁语词表

## 附：禁语词表使用说明

1. **命中即高风险（红线词，1-34号）**：自动标记为"高风险"，建议阻断使用，必须人工复核后方可使用
2. **命中需复核（高风险词，35-55号）**：标记为"高风险"，需修改后重新提交预检
3. **建议修改（中风险词，56-63号）**：标记为"中风险"，建议补充或修改后使用
4. **措辞建议（低风险词，64-68号）**：标记为"低风险"，可自行调整后使用
5. 本词表应定期更新，跟踪监管新规和行业违规案例
