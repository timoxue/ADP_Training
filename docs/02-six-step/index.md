# 方法篇：六步分析法

六步分析法是本培训的核心方法论，贯穿所有场景实战。

## 六步流程

1. **定义场景** — 明确"谁用、做什么、限制是什么"
2. **选择模式** — 根据流程复杂度选 ADP 应用模式
3. **设计知识库** — 确定 RAG 所需文档与切分策略
4. **编写提示词** — 按框架（角色/风格/输出/限制）结构化
5. **配置工具/工作流** — 连接外部数据源和自动化节点
6. **评测与发布** — 对话调试 → 评测 → 发布渠道
