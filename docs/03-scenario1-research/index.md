# 场景一：投研自动化

## 场景描述

> 让研究员把问题抛给 AI，AI 扫遍研报库给出答案 + 来源片段。

**痛点**：研报库数量大，人工检索耗时；关键数据散落在 PDF 不同位置。  
**方案**：标准模式 + 知识库（研报 PDF 导入）+ 大模型知识问答节点

## 产出物

- 研报问答应用（可发布到微信/企业微信）
- 结构化数据抽取工作流

## 涉及 ADP 功能

- 知识库：PDF 批量导入、智能切分
- 标准模式：角色指令 + 知识库绑定
- Widget：结果卡片展示
