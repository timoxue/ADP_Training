# 证券服务合同模板（含关键条款标注）

> 本文档提供证券行业常见合同的标准模板，标注了合规预检需重点关注的关键条款，供合规预检助手检索和比对使用。

---

---

### 第一条 服务内容

乙方为甲方提供以下证券投资顾问服务：
1. 证券投资建议服务，包括但不限于行业分析、个股研究、资产配置建议；
2. 投资策略咨询服务；
3. 定期投资报告和风险提示。

**【合规标注：服务内容必须与经营许可证范围一致，不得超范围经营】**

### 第二条 服务期限

本合同有效期为 ____ 年，自 ____年__月__日 起至 ____年__月__日 止。

**【关键条款-有效期】预检需提取：合同起止日期**

### 第三条 服务费用

1. 甲方应向乙方支付服务费用，标准为：__________；
2. 支付方式：__________；
3. 支付时间：__________。

**【合规标注：服务费用必须通过机构账户收取，不得以个人名义收取（《证券投资顾问暂行规定》第15条）】**

### 第四条 甲方的权利与义务

1. 甲方有权自主作出投资决策，乙方不得代理甲方进行证券交易；
2. 甲方有权了解投资建议的依据和分析方法；
3. 甲方应当如实填写风险评估问卷，配合乙方进行适当性评估；
4. 甲方应当自行承担投资风险和损失。

**【关键条款-违约责任】预检需确认：是否明确"甲方自主决策、自行承担风险"**
**【合规标注：不得有代客操作条款，不得有收益分成条款】**

### 第五条 乙方的权利与义务

1. 乙方应当根据甲方的风险承受能力提供适当的投资建议；
2. 乙方不得以任何方式承诺或保证投资收益；
3. 乙方不得代理甲方作出投资决策；
4. 乙方应当向甲方说明投资建议的依据；
5. 乙方应当对甲方的投资信息严格保密。

**【关键条款-保密条款】预检需提取：保密义务的具体描述**

### 第六条 风险提示

1. 证券投资存在风险，甲方可能遭受本金损失甚至全部损失；
2. 乙方提供的投资建议仅供参考，不构成投资决策的最终依据；
3. 历史业绩不代表未来收益；
4. 甲方应根据自身风险承受能力审慎决策。

**【合规标注：风险提示条款必须显著标注，不得以过小字体或模糊方式呈现】**

### 第七条 免责条款

1. 乙方不对投资结果承担任何责任，甲方自行承担投资风险；
2. 因不可抗力导致的损失，双方互不承担责任；
3. 因甲方未按乙方建议操作导致的损失，乙方不承担责任。

**【关键条款-赔偿帽】预检需确认：是否有赔偿上限条款，赔偿范围是否合理**

### 第八条 违约责任

1. 任何一方违反本合同约定，应向守约方支付违约金 ______ 元；
2. 违约金不足以弥补损失的，违约方还应赔偿差额部分；
3. 赔偿总额不超过甲方已支付服务费用的 ___ 倍。

**【关键条款-违约责任】预检需提取：违约金金额、赔偿上限倍数**

### 第九条 争议解决

1. 本合同履行过程中发生争议的，双方应首先协商解决；
2. 协商不成的，任何一方均可向 **__________人民法院** 提起诉讼。

**【关键条款-管辖法院】预检需提取：管辖法院名称和所在地**

### 第十条 合同变更与解除

1. 经双方协商一致，可以变更或解除本合同；
2. 任何一方提前解除合同的，应提前 ____ 日书面通知对方；
3. 解除后，已收取但未提供服务对应部分的费用应退还甲方。

### 第十一条 其他约定

1. 本合同未尽事项，双方可签订补充协议；
2. 本合同一式两份，双方各执一份，具有同等法律效力。

**甲方（签字/盖章）：** ________________  **日期：** ________
**乙方（盖章）：** ________________  **日期：** ________

## 附：合同关键条款预检清单

| 关键条款 | 模板一位置 | 预检要点 |
|---|---|---|
| 有效期 | 第二条 | 提取合同起止日期，确认服务期限合理 |
| 服务费用 | 第三条 | 确认费用收取方式为机构账户，非个人名义 |
| 代客操作禁止 | 第四条第1款 | 确认有明确禁止代客操作条款 |
| 风险提示 | 第六条 | 确认风险提示显著、完整、无规避 |
| 赔偿帽 | 第七条/第八条 | 提取赔偿上限，确认赔偿范围合理 |
| 违约责任 | 第八条 | 提取违约金金额和赔偿倍数 |
| 管辖法院 | 第九条 | 提取管辖法院名称和所在地 |
| 保密条款 | 第五条第5款 | 确认有保密义务条款 |
| 收益禁止承诺 | 第五条第2款 | 确认有"不得承诺收益"条款 |
| 适当性义务 | 第五条第1款 | 确认有适当性评估条款 |
