# AI 应用实战培训

> 腾讯云智能体开发平台 · 证券行业应用实战手册

欢迎参加本次实战培训！本手册面向证券行业的技术与业务人员，通过 **四个真实场景** 带你从零搭建 AI 应用。

---

## 课程地图

```
基础篇 → 方法篇 → 场景实战（4个）→ 进阶篇
```

| 章节 | 内容 | 难度 |
|------|------|------|
| [基础篇](01-foundation/index.md) | LLM/RAG/Agent 通用知识 + ADP 平台功能 + 四种模式 | ⭐ |
| [方法篇](02-six-step/index.md) | 六步分析法 · 应用创建与提示词实战 | ⭐⭐ |
| [场景一](03-scenario1-research/index.md) | 投研自动化 · 研报问答与数据抽取 | ⭐⭐ |
| [场景二](04-scenario2-advisor/index.md) | 财富投顾 · 7×24 助手守住适当性红线 | ⭐⭐ |
| [场景三](05-scenario3-compliance/index.md) | 合规风控 · "逐字核对"变自动化流水线 | ⭐⭐⭐ |
| [场景四](06-scenario4-post-invest/index.md) | 投后经营 · 公告跟踪与简报自动推送 | ⭐⭐⭐ |
| [进阶篇](07-multi-agent/index.md) | Multi-Agent · 搭一个"投研小团队" | ⭐⭐⭐⭐ |

## 学完你能做什么

- ✅ 独立创建四种模式的智能体应用
- ✅ 设计符合监管要求的合规提示词框架
- ✅ 对接知识库、工作流、工具插件
- ✅ 搭建投研 Multi-Agent 自动化流水线

!!! tip "建议学习路径"
    初次接触 → 从**基础篇**开始顺序阅读  
    有 AI 背景 → 直接跳到**方法篇**，再选择感兴趣的场景  
    只看某个场景 → 每个场景章节均可独立阅读
