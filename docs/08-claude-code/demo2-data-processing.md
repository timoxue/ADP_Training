## 前置准备

安装 Python 分析依赖（仅需执行一次）：

```bash
pip install pandas matplotlib plotly openpyxl
```

---

## 操作步骤

### 步骤 1：启动 Claude Code

```bash
claude
```

### 步骤 2：快速了解数据概况

```
读取当前目录的 quarterly_data.csv，告诉我：
1. 数据包含哪些公司、哪些时间段
2. 哪家公司的营收增速最快
3. 哪家公司的净利润率最高
```

**预期输出**：

> 数据集包含 5 家证券公司 2023Q4 至 2024Q3 共 5 个季度的财务数据。
>
> **营收增速最快**：华海证券，2024Q2 营收较 2023Q4 增长约 21%，增速居首。
>
> **净利润率最高**：华海证券，2024Q2 净利润率达 23.4%；东盛证券紧随其后，2024Q2 净利润率 22.6%。

---

### 步骤 3：生成竞争格局分析报告

```
基于这份数据，帮我写一份竞争格局分析报告，结构：
1. 行业整体趋势（2024年前三季度）
2. 各公司综合排名（营收 / 利润 / ROE 三个维度分别排名）
3. 各公司的核心优势与薄弱业务线
4. 结论与启示

用 Markdown 格式，语气专业
```

**预期输出**（排名部分）：

| 排名 | 营收规模 | 净利润 | ROE |
|------|---------|-------|-----|
| 1 | 华海证券（221,000万） | 华海证券（51,800万） | 华海证券（2.8%） |
| 2 | 东盛证券（172,000万） | 东盛证券（38,900万） | 东盛证券（2.4%） |
| 3 | 远洋投资（118,000万） | 远洋投资（24,800万） | 远洋投资（2.4%） |
| 4 | 金桥财富（89,000万） | 金桥财富（17,600万） | 金桥财富（2.1%） |
| 5 | 鸿达资本（68,000万） | 鸿达资本（12,400万） | 鸿达资本（1.8%） |

### 步骤 4：生成可视化图表

```
帮我写一个 Python 脚本，读取 quarterly_data.csv，生成以下图表并保存为 HTML 文件：
1. 各公司2024年三季度营业收入对比柱状图
2. 华海证券四季度净利润趋势折线图
3. 2024Q3各公司业务收入结构饼图（经纪/自营/资管/投行）

使用 plotly 库，图表标题用中文，保存为 analysis_charts.html
```

Claude Code 生成脚本后，继续输入：

```
请运行刚才生成的脚本
```

查看结果：

```bash
# Windows
start analysis_charts.html

# Mac
open analysis_charts.html
```
