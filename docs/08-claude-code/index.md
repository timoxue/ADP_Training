# Claude Code 实战演示

> 三个场景 · 面向证券公司职能部门

---

## 什么是 Claude Code？

Claude Code 是 Anthropic 推出的命令行 AI 工具，可以直接读取本地文件、执行代码、处理数据，完成从分析到生成的完整任务链——不需要手动复制粘贴，不需要切换工具。

## 快速开始

确认 Claude Code 已安装：

```bash
claude --version
```

进入演示目录后启动：

```bash
claude
```
