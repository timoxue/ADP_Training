# 框架演示 · OpenClaw & Hermes

> 开源 Agent 框架现场演示手册

---

## 两个框架，两种能力

| 框架 | 核心定位 | 演示亮点 |
|------|---------|---------|
| **OpenClaw（龙虾）** | 技能体系 + 常驻 Gateway | 现场写一个证券合规预检技能，3 分钟上手 |
| **Hermes Agent** | 四层分层记忆 | 关掉重开，它还认识你 |

---

## 适合什么场景

- **OpenClaw**：适合需要把能力沉淀下来、在团队里复用的场景。技能文件一次写好，全团队共享。
- **Hermes**：适合需要持续跟进、跨会话连续工作的场景。投研分析、客户跟进、项目管理。

---

## 演示顺序建议

1. 先演示 OpenClaw（展示技能挂载）
2. 再演示 Hermes（展示跨会话记忆）
3. 最后做对比总结：两者解决不同问题，互补不竞争
