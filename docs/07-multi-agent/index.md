# 进阶篇：Multi-Agent

> **搭一个"投研小团队"，多线程处理复杂投研任务**

单个智能体搞不定的复杂投研任务，交给"一个小团队"。

## 团队架构

```
主 Agent（调度/接活/分工/汇总）
├── 资料检索 Agent    — 数据采集/公告解读
├── 数据分析 Agent    — 指标计算/估值建模
├── 风控合规 Agent    — 合规扫描/风险提示
└── 结论撰写 Agent    — 报告生成
```

## 验收标准

任务能在多个子 Agent 间流转，并产出经过交叉校对的综合结论。

## 最常见问题

> **忘记配置转交关系** → 任务无法流转，主 Agent 自己全做了
