# 基础篇概览

本篇帮助你建立必要的背景知识——既包括通用的 AI 原理，也包括 ADP 平台的具体设计逻辑。

## 本篇内容

- **LLM · RAG · Agent**：大模型是什么、检索增强生成解决什么问题、Agent 比普通 AI 多了什么
- **主流模型与框架速览**：国内外主流模型对比、五类 Agent 框架的定位与选型
- **ADP 平台功能总览**：核心模块、六大治理机制、关键配置参数
- **四种应用模式**：Claw / 标准 / 单工作流 / Multi-Agent 的选择逻辑
- **Skills 与提示词框架**：技能体系原理、两套提示词框架、平台变量
- **推荐 Skills 分类**：按场景整理的证券行业推荐技能组合
